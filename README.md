# 投资者行为研究

本仓库为投资者行为研究的公开资源库，探索金融市场中观察到的行为模式、决策过程与行为倾向。

## 仓库结构

| 文件夹 / 文件 | 说明 |
|---|---|
| `documents/` | 原始来源文档、报告及参考资料，作为研究输入使用 |
| `processed_documents/` | 经过清洗、标注和结构化处理后的来源文档 |
| `images/` | 研究数据生成的可视化输出，包括图表、图形和图示 |
| `scripts/` | 数据处理与分析脚本 |
## 快速开始

如尚未安装 `uv`，请先执行：

```bash
pip install pipx
pipx install uv
```

然后安装项目依赖：

```bash
uv sync
```
